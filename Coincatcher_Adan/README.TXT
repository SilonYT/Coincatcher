PROJECT TITLE: CoinCatcher 
PURPOSE OF PROJECT: LB2 
VERSION: Version 1.0 
AUTHORS: Adan Mohamed 

USER INSTRUCTIONS:
  	
	Introduction:   

- Dieses Spiel ist ein einfaches 2D-Spiel, das in der Programmiersprache Java geschrieben wurde und auf der Greenfoot-Plattform ausgeführt wird. Es geht darum, Punkte zu sammeln, während man Hindernissen ausweicht.

	Systemvoraussetzungen:

- Greenfoot 3.5.0 oder höher
- Java 8 oder höher

    	Installation:

- Greenfoot von der offiziellen Website herunterladen und installieren: https://www.greenfoot.org/download
- Das Spiel herunterladen und entpacken.
- Das Spiel in Greenfoot öffnen und ausführen.

    Spielbedienung:

- Verwenden Sie die Pfeiltasten, um sich nach links oder rechts zu bewegen.
- Sammeln Sie so viele Punkte wie möglich, indem Sie sie mit dem Spielcharakter berühren.
- Vermeiden Sie Hindernisse, um zu verhindern, dass das Spiel endet.
- Wenn Sie 350 Punkte erreichen, wird das Spiel automatisch beendet und es wird eine GameOver-Anzeige angezeigt.

    Code-Informationen:

- Die Klasse Punkte verwaltet die Punktezählung und zeichnet sie auf dem Bildschirm.
- Die Klasse GameOverScreen zeigt die GameOver-Anzeige an, wenn das Spiel endet.

	Spiel Start:
- Beim Starten Wählen Sie ein Level aus Level 1 oder Level 2
- Level 1 hat nur ein Gegner da wo Level 2 (2 Gegner hat)
